# 第一章 绪论

## 1.1 研究背景与意义

能源与环境问题是当今社会可持续发展面临的重大挑战，发展清洁、高效的电化学能源转化与存储技术已成为各国科技竞争的焦点。燃料电池、金属—空气电池、电解水制氢以及有机小分子电氧化等电化学能源器件，均依赖电极表面发生的电催化反应实现化学能与电能之间的相互转化，而这些反应的效率在很大程度上取决于电催化剂的本征活性、稳定性以及活性位点的数量[1,4]。因此，开发高活性、高稳定且成本可控的电催化材料，是推动相关能源技术规模化应用的关键，也是电化学与材料科学交叉领域长期关注的核心课题。

长期以来，铂（Pt）、钌（Ru）、铱（Ir）等贵金属及其氧化物因具有优异的本征催化性能，被广泛用作析氢反应（HER）、氧还原反应（ORR）和析氧反应（OER）的基准催化剂[2,3]。然而，贵金属资源储量有限、价格高昂，难以满足大规模工业应用对低成本和长周期稳定运行的要求，这一矛盾促使研究者将目光转向储量丰富、价格低廉的过渡族金属材料。过渡族金属元素具有未充满的 d 电子轨道，价态可变，电子结构调控空间大，在多种电催化反应中表现出良好的应用潜力，被认为是替代贵金属的重要方向[4,6]。

在此基础上，将致密金属进一步多孔化能够显著改善其电催化性能。一方面，多孔结构大幅增加了材料与电解液的接触面积，提高了活性位点的暴露程度；另一方面，其三维连通的金属骨架与开放孔道为电子传输和反应物、产物的传质提供了便捷通道[4,6]。与需要导电剂和粘结剂连接的纳米颗粒催化剂相比，多孔金属合金兼具金属骨架的导电连续性与高比表面积的优势，避免了额外界面电阻的引入。因此，多孔过渡族金属合金材料在兼顾催化活性、活性位点数量与材料成本方面具有独特优势，对降低电催化体系对贵金属的依赖、推动清洁能源技术的发展具有重要的理论意义与应用价值。深入研究其制备方法、孔结构演化规律及其对电催化行为的影响，对于理解构效关系、指导高性能电极材料的设计具有重要意义。

## 1.2 电化学催化基础与反应机理

电化学催化是指在电极与电解液界面上、由电极电位驱动并借助催化剂降低反应活化能的多相催化过程[8]。与普通多相催化相比，电催化反应的速率不仅受温度、反应物浓度等因素影响，更可通过改变电极电位加以调控：电位的变化直接改变界面电场强度和电子转移的驱动力，从而在较宽范围内连续调节反应速率，这是电催化区别于一般化学催化的本质特征。电极与电解液接触时，界面处会形成双电层，反应物的吸附、电荷转移和产物脱附等基元步骤均在此区域内完成。根据电子转移是否经由电极表面吸附的中间体，电催化反应可分为外层电子转移过程和内层电子转移过程；燃料电池电氧化、有机小分子氧化等大多数电催化反应属于内层过程，其电子转移与反应物的化学吸附紧密耦合，因而对催化剂的表面组成和原子排列高度敏感[8]。

一个完整的电催化反应通常包含若干串联步骤：反应物由本体溶液向电极表面的传质、反应物在活性位点上的吸附、界面电荷转移与表面化学转化、以及产物的脱附和向本体溶液的扩散。其中，含氧或含碳中间体在催化剂表面的吸附往往是决定反应路径和速率的关键环节。整个过程的快慢可由若干电化学参数加以表征：过电位（η）反映了驱动反应实际进行所需的额外电位，是衡量催化剂能耗的重要指标；交换电流密度（j₀）表征平衡电位附近的本征反应速率；塔菲尔（Tafel）斜率则可用于推断反应的速控步骤与反应机理；在给定电位下的电流密度直接反映了电极的法拉第电流响应，常用于不同催化剂之间催化活性的横向比较[8]。

从微观层面看，电催化活性的高低主要取决于反应中间体在催化剂表面的吸附自由能。根据 Sabatier 原理，理想的催化剂应使中间体的吸附既不过强也不过弱：吸附过弱则反应物难以活化，吸附过强则中间体或产物难以脱附而毒化表面，二者均不利于反应进行。将催化活性对吸附能作图，往往呈现"火山型"曲线，处于火山顶点附近的材料具有最优活性[3,8]。对于金属及合金催化剂而言，其对中间体的吸附强度与表面原子的电子结构密切相关，可用 d 带中心理论加以描述：d 带中心相对费米能级的位置越高，金属对吸附质的成键越强。由此，通过合金化引入异种金属，可借助配位效应（电子结构改变）与应变效应（晶格畸变）调节 d 带中心，从而优化中间体吸附、提升本征活性，部分体系还可形成双功能机制，由不同组元分别承担吸附与脱附等功能[1,4]。

此外，活性位点的性质同样影响催化行为。台阶、棱边、缺陷、晶界以及低配位的表面原子通常具有更高的反应活性，其数量与暴露程度直接决定了电极的整体催化能力。综上，提升电催化性能的途径主要有二：一是提高活性位点的本征活性，可通过合金化等手段调控电子结构实现；二是增加可参与反应的活性位点数量并改善传质条件，可通过构筑多孔结构实现。这一认识为后续多孔合金及铜基催化材料的研究提供了理论依据。

## 1.3 多孔过渡族金属合金电催化材料研究现状

多孔过渡族金属合金材料的研究经历了从形貌构筑到结构—性能关系分析的发展过程。早期研究多关注多孔结构能否成功形成以及材料比表面积是否提高；随着研究的深入，研究者逐渐认识到多孔结构的作用并不只是增加表面积，还涉及孔道连通性、骨架稳定性、局部传质环境与表面电子结构等多方面因素[4,6]。下面分别从多孔结构的电催化作用、制备方法和表征手段三个方面对其研究现状加以综述。

### 1.3.1 多孔结构的电催化作用与构效关系

比表面积的增加通常意味着电解液可接触的活性位点增多，但物理比表面积并不完全等同于电化学有效面积（ECSA）。部分孔道可能因尺寸过小或连通性不足而无法被电解液充分浸润，部分位点也可能因传质受限而难以有效参与反应，因此孔径分布、孔道连通性与骨架连续性需要综合考虑[6]。三维连通的金属骨架能够保持良好的电子传输能力，开放孔道有利于反应物进入和产物脱离，丰富的表面缺陷与晶界则可提供更多低配位活性位。在孔径层次上，兼具大孔、介孔与微孔的层级多孔结构往往更为有利：大孔有助于电解液的快速渗透与气泡逸出，介孔和微孔则贡献了主要的活性比表面积，二者协同可在保证传质的同时最大化活性位点的暴露。研究表明，孔结构与合金化效应往往共同决定催化剂的实际电流响应与长期稳定性[4,6]。

值得注意的是，多孔材料在反应过程中并非一成不变。长期运行中可能发生骨架粗化、金属溶出、表面氧化与结构重构等现象，进而引起活性位点数量和电子结构的变化。因此，仅依靠初始形貌图像难以完整解释材料行为，需结合反应过程中及反应后的结构演化与产物分析，才能准确揭示其构效关系与性能来源。

### 1.3.2 多孔合金的制备方法

多孔金属合金的制备方法较为多样，常见方法包括去合金化法、模板法、电沉积法、热处理还原法以及阳极氧化法等。

去合金化法通过选择性溶解合金中较活泼的组元，使剩余组元在界面处发生扩散和重排，从而形成三维双连通的孔道与韧带结构[5,15]。该方法工艺相对简单、适用合金体系较广、对孔结构的调控较为直接，且通常无需复杂模板与高温处理，特别适合金属片状电极的表面改性。通过调节腐蚀介质组成、浓度、温度和时间，可在较宽范围内调控韧带尺寸与孔径，是制备纳米多孔金属最常用的途径之一[15]。

模板法则借助硬模板或软模板的空间限域作用对孔结构进行复制与诱导，其突出特点是孔径与形貌易于预设，尤其适合构筑介孔纳米线、中空纳米颗粒和层级孔载合金体系，但模板去除、表面残留与规模化成本仍是其限制因素。电沉积法能够在导电基底上直接形成合金电极，具有工艺温和、厚度可控和界面接触良好的特点。热处理还原法适合将金属盐、金属有机框架或配位聚合物转化为多孔合金/碳复合催化剂。阳极氧化法则可在金属表面构筑自有序的多孔氧化层，已被用于制备高耐久性的多孔电催化电极[2]。不同制备路线得到的孔结构、元素分布与骨架稳定性各异，最终会对电催化行为产生显著影响[6]。

### 1.3.3 多孔结构的表征方法

多孔合金材料的结构表征通常需要多种手段相互配合。扫描电子显微镜（SEM）和透射电子显微镜（TEM）用于观察微米至纳米尺度的表面形貌、孔道分布与骨架完整性；氮气吸脱附（BET）法用于测定比表面积与孔径分布；能谱（EDS）用于分析元素组成及其分布；X 射线衍射（XRD）用于确定物相组成与晶体结构变化[6,7]。在电化学层面，还可通过双电层电容法估算电化学活性面积，以区分物理比表面积与实际参与反应的有效面积。只有将形貌、物相、成分与电化学有效面积等信息结合起来，才能较为全面地揭示多孔结构与电催化性能之间的内在联系[6]。现阶段研究仍存在不足之处：部分工作较关注初始催化性能，对反应后孔结构演化、金属溶出和骨架粗化的分析不够；部分工作对物理孔结构与电化学活性面积之间的区分不足，容易将形貌改善直接等同于性能提升。

## 1.4 铜基电催化材料研究现状

在众多过渡族金属中，铜（Cu）因其特殊的电子结构和突出的成本优势而受到广泛关注。铜的价电子构型为 3d¹⁰4s¹，地壳丰度高、价格低廉、导电性优良，且具有可变价态，在电化学环境中可发生丰富的氧化还原转变[14]。在碱性介质中，铜电极表面随电位升高依次发生 Cu(0)→Cu(I)→Cu(II)→Cu(III) 的转变，相应地生成 Cu₂O、CuO/Cu(OH)₂ 以及高价的 CuOOH 等表面物种。其中，强氧化性的 Cu(III)（如 CuOOH 或 Cu(OH)₄⁻）被认为是媒介有机小分子氧化的关键活性物种，这一表面氧化还原特性使铜基材料在多种电催化反应中表现出独特的活性[14]。

得益于上述性质，铜基催化材料已被广泛应用于析氢反应、二氧化碳电还原（对 C₂₊ 等多碳产物具有较高选择性）、有机小分子（甲醇、乙醇、葡萄糖、甲醛等）电氧化以及电化学传感等领域[11,14]。然而，单一铜电极在本征活性和长期稳定性方面仍有不足，反应过程中易发生表面重构、氧化层增厚和金属溶出，导致催化活性衰减。

为克服上述问题，研究者发展了多种改性策略。其一是纳米化与多孔化：通过去合金化等方法制备纳米多孔铜，可显著增大比表面积、暴露更多低配位活性位，已被证明对有机小分子等反应具有良好的电催化活性[15,16]。其二是合金化与双金属化：在铜中引入锡（Sn）、锌（Zn）、镍（Ni）、银（Ag）等元素形成合金，可借助配位效应与应变效应调节铜的电子结构、改变表面氧化层的生成行为，并通过双功能机制提升活性与稳定性[11]。以铜锡（Cu-Sn）体系为例，锡的引入不仅能够改变铜基材料的表面电子结构和化学稳定性，其自身在电化学环境中也可能影响表面氧化层的形成及有机中间体的吸附行为。此外，铜锡青铜本身是一类典型的可去合金化体系，通过选择性溶解其中较活泼的组元，可在保留铜锡骨架的同时构筑多孔结构，从而将合金化与多孔化的优势结合于同一电极，为铜锡基多孔电极的制备提供了便利。其三是氧化物/氢氧化物衍生策略，通过预氧化再还原等手段构筑富缺陷、富晶界的活性表面。综合来看，将多孔化与合金化相结合，是同时提升铜基电极活性位点数量与本征活性的有效途径，也是当前铜基电催化材料研究的重要方向。

## 1.5 甲醛电催化氧化机理及研究现状

### 1.5.1 甲醛电氧化反应机理

甲醛（HCHO）是结构最简单的醛类有机小分子，其电催化氧化过程涉及分子吸附、脱氢、中间体转化与产物生成等环节。相较于复杂有机分子，甲醛体系反应路径较为清晰，常被用作研究金属电极表面含氧有机小分子吸附与氧化行为的模型反应。在水溶液尤其是碱性介质中，甲醛主要以水合形式甲二醇 [CH₂(OH)₂] 及其去质子产物甲二醇阴离子 [CH₂(OH)O⁻] 的形态存在，这一存在形态对其在电极表面的吸附与活化具有重要影响[9]。

在碱性条件下，铜、镍、钴等过渡金属电极对甲醛的氧化通常遵循高价金属物种媒介的间接氧化机制。以铜为例，电极表面随电位升高发生氧化还原转变并生成高价活性物种：

```
Cu + 2OH⁻ → Cu(OH)₂ + 2e⁻
Cu(OH)₂ + OH⁻ → CuOOH + H₂O + e⁻      （Cu(II) → Cu(III)）
```

随后，强氧化性的 Cu(III)（CuOOH）将吸附态甲醛氧化，自身被还原为低价铜物种，形成表面氧化还原媒介循环：

```
CuOOH + CH₂(OH)O⁻ → Cu(OH)₂ + HCOO⁻ + …
```

甲醛电氧化的主要液相产物为甲酸盐（HCOO⁻），并可能伴随少量 CO、CO₂ 等深度氧化产物。与此同时，甲醛在碱性介质中还可经脱氢途径释放氢气：

```
HCHO + OH⁻ → HCOO⁻ + H₂↑
```

正因如此，甲醛阳极氧化常被用于与阴极析氢相耦合，以降低体系所需电压、实现"双氢"产出[10,11]。然而，甲醛电氧化仍面临过电位较高、表面易被类 CO 中间体毒化、电荷转移动力学缓慢等挑战，其活性高度依赖于电极表面高价活性物种的生成能力及活性位点的结构[9,10]。

### 1.5.2 甲醛电氧化催化材料研究现状

早期对甲醛电氧化的研究多集中于铂（Pt）、钯（Pd）、金（Au）等贵金属电极。借助单晶电极，研究者揭示了甲醛氧化的结构敏感性与反应路径，例如金单晶表面甲醛氧化伴随析氢、并受表面晶面结构和铊欠电位沉积的影响。贵金属电极虽然活性较高，但成本昂贵且易受中间体毒化，限制了其实际应用。为兼顾活性与成本，钯与铜、银等 IB 族金属形成的合金也被用于甲醛及甲酸盐的电氧化研究，以期在较弱碱性条件下促进甲醛的深度氧化[17]。

近年来，非贵金属及其复合、合金材料成为研究重点。在铜基材料方面，研究者系统考察了甲醛在铜单晶电极上的电氧化行为[9]，并发展了氯化铜修饰铜电极、铜—导电聚合物复合电极等体系以提高活性；在双金属与掺杂方面，低镍掺杂铜电极通过 Cu⁰ 与 Cu⁺ 的协同作用，分别促进反应能量降低与 C–H 键断裂，从而提升了甲醛氧化与析氢性能[10]，铜—银双金属电催化剂则实现了甲醛氧化与水还原相耦合的高效"双氢"产出[11]。在结构调控方面，电沉积构筑的结构化铜催化剂在碱性介质中对甲醛氧化表现出良好活性[12]。镍基材料同样备受关注，NiOOH 是甲醛氧化的有效活性物种，镍/沸石、镍—锰磷酸盐等体系均显示出可观的氧化电流密度[13]。总体而言，提高比表面积、构筑多孔结构并辅以合金化调控，是增强甲醛电氧化活性、缓解表面毒化的主要趋势。

从应用角度看，甲醛电氧化兼具环境与能源双重价值：一方面有助于含甲醛废水中污染小分子的转化与处理；另一方面，将其作为低电位阳极反应与阴极析氢耦合，可降低电解能耗。然而，目前研究对铜基合金多孔结构与甲醛电氧化性能之间的定量关联、以及反应后结构演化与液相产物的系统分析仍显不足。因此，如何通过可控的多孔化处理调控铜基合金的孔结构，并建立其与甲醛电氧化性能之间的关联，仍是该领域有待深入探讨的问题。

## 1.6 本课题主要研究内容

针对上述问题，本课题以 FeCl₃ 辅助腐蚀制备多孔 Cu-Sn 合金及其甲醛电氧化性能为研究主题，围绕"腐蚀条件—多孔结构—电化学性能"三者之间的关系展开，旨在通过可控腐蚀获得不同多孔结构的铜锡合金电极，并揭示其孔结构与甲醛电氧化行为之间的内在联系，为铜锡基多孔合金电极材料的设计提供实验依据。

研究对象为 Cu-6%Sn 合金片，腐蚀体系为 1 mol/L HCl 与 FeCl₃ 组成的混合溶液。其中 HCl 提供酸性溶解环境，FeCl₃ 作为氧化性组分诱导合金表面组分选择性溶出与重构，二者协同实现表面多孔化（去合金化）。实验固定 HCl 浓度为 1 mol/L，通过改变 FeCl₃ 浓度（0.05、0.10、0.15 mol/L）和腐蚀时间（30、60、90 min）制备九组不同腐蚀程度的样品；横向比较同一腐蚀时间下不同 FeCl₃ 浓度的样品，可分析氧化性组分浓度对腐蚀程度的影响，纵向比较同一浓度下不同腐蚀时间的样品，可分析腐蚀时间对多孔结构演化的影响。腐蚀前，合金片经 400、800、1200 目水砂纸逐级打磨、无水乙醇超声清洗并冷风干燥，以保证不同样品初始表面状态一致、结果可比[5]。

在结构表征方面，采用扫描电子显微镜（SEM）观察腐蚀后样品的表面形貌、孔结构形成情况、孔道分布及骨架完整性，判断腐蚀处理是否成功构筑多孔结构；采用 X 射线衍射（XRD）分析腐蚀前后样品的物相组成与晶体结构变化，考察合金主体相是否保持稳定及是否生成新相。SEM 与 XRD 结果相互结合，用于评价不同腐蚀条件下样品表面多孔化与骨架稳定性之间的平衡[6,7]。

在电化学性能测试方面，采用三电极体系，以制备的多孔 Cu-Sn 合金样品为工作电极，铂（Pt）片为对电极，Hg/HgO 电极为参比电极。首先在 1 mol/L NaOH 溶液中进行循环伏安（CV）活化，使电极表面达到相对稳定状态并形成与甲醛氧化相关的活性位；随后在 1 mol/L NaOH + 0.5 mol/L 甲醛溶液中进行 CV 与线性扫描伏安（LSV）测试，通过氧化峰电流、起始氧化电位、特定电位下的电流密度及循环稳定性等指标评价样品的甲醛电氧化催化性能。最后，对反应后的电解液采用核磁共振（NMR）进行液相产物分析，以判断高电流响应样品是否确实促进了甲醛的转化，从而将电流响应与产物变化联系起来，更准确地说明催化性能的来源。

综上，本课题拟解决的核心问题包括三个方面：（1）FeCl₃ 浓度与腐蚀时间如何影响 Cu-6%Sn 合金表面多孔结构的形成；（2）不同多孔结构样品在碱性甲醛溶液中的电化学氧化行为是否存在差异；（3）腐蚀制备形成的多孔结构与甲醛电氧化性能之间是否存在可解释的关联。围绕上述问题，本论文后续章节将依次介绍实验方案（材料制备、结构表征与电化学测试方法）、实验结果与讨论（表面形貌分析、物相分析、电化学性能评价及结构—性能关系讨论），并在此基础上总结全文、提出展望，以期为铜锡基多孔合金电催化材料的制备与性能优化提供参考[5,6]。
